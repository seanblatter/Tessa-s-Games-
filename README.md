# 💝 Tessa's Games 💝

A beautiful, mobile-friendly web application featuring Wordle, Sudoku, plus three 9x9 pattern games customized for Tessa.

## 🎮 Games Included

### Wordle
- Guess special 5-letter words customized for Tessa
- Classic Wordle gameplay with color-coded feedback
- On-screen keyboard for mobile devices
- Physical keyboard support

### Sudoku (9x9)
- Classic 9x9 number puzzle gameplay
- Touch-friendly number pad input
- Auto-check support and keyboard navigation

### Pixel Patterns (9x9)
- Recreate a hidden 9x9 target pattern
- Tap cells to toggle individual pixels
- Use quick target peek for memorization practice

### Lights Out 9x9
- Cross-toggle mechanics (selected cell + orthogonal neighbors)
- Match a hidden 9x9 target layout
- Designed for short logic bursts

### Mirror Mosaic (9x9)
- Every tap mirrors to the opposite side
- Build symmetrical patterns on a 9x9 board
- Visual puzzle with spatial strategy

## 🚀 Getting Started

### Running Locally

Simply open `index.html` in your web browser, or use a local server:

```bash
python3 -m http.server 8080
```

Then navigate to `http://localhost:8080` in your browser.
